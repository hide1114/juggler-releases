プライバシーポリシー

最終更新日: 2025年11月30日

ジャグラー設定推測（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に最大限の注意を払っています。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明するものです。

1. 収集する情報

1.1 個人情報

本アプリは、ユーザーの個人情報を一切収集しません。

  • 氏名、メールアドレス、電話番号などの個人を特定できる情報は収集しません
  • 位置情報は収集しません
  • 連絡先情報は収集しません

1.2 利用データ

本アプリは、以下のデータをローカル（端末内）でのみ処理します：

  • 総回転数、BIG回数、REG回数、ブドウ回数などの入力データ
  • 設定推測の計算結果
  • アプリの設定情報（同意状況など）
重要

これらのデータは、すべて端末内でのみ保存・処理されます。外部サーバーへの送信は一切行いません。

2. 情報の使用目的

本アプリで処理されるデータは、以下の目的でのみ使用されます：

  • 統計的な設定推測計算の実行
  • 推測結果の表示
  • アプリの基本機能の提供

3. 情報の保存場所

すべてのデータは、ユーザーの端末内にのみ保存されます：

  • データは端末のローカルストレージ（AsyncStorage）に保存されます
  • 外部サーバー、クラウドサービスへのデータ送信は行いません
  • アプリをアンインストールすると、すべてのデータが削除されます

4. 第三者へのデータ提供

本アプリは、ユーザーのデータを第三者に提供、共有、販売することは一切ありません。

5. 広告について

本アプリは広告を表示する場合があります：

  • 広告配信事業者が、広告配信の最適化のために限定的な情報（デバイス情報、広告ID）を収集する可能性があります
  • これらの情報収集は、各広告事業者のプライバシーポリシーに従って行われます
  • ユーザーは端末の設定から、広告トラッキングをオプトアウトすることができます

主要な広告事業者のプライバシーポリシー

  • Google AdMob: https://policies.google.com/privacy

6. アプリのアップデート確認

本アプリは、新しいバージョンの確認のため、以下の情報のみにアクセスします：

  • GitHub Releases API（公開情報）
  • 現在のアプリバージョン番号

この際、ユーザーの個人情報や利用データは送信されません。

7. データのセキュリティ

本アプリは、データのセキュリティ確保のため、以下の対策を講じています：

  • すべてのデータ処理を端末内で完結
  • 外部通信の最小化（バージョン確認のみ）
  • 不要な権限の要求を行わない

8. お子様のプライバシー

本アプリは20歳未満の方の利用を想定していません。保護者の方は、お子様が本アプリを使用しないよう管理をお願いいたします。

9. 免責事項

本アプリは以下の点についてご了承ください
  • 本アプリは、パチスロメーカーとは一切関係のない非公式ツールです
  • 推測結果の正確性を保証するものではありません
  • 本アプリの使用によって生じた損害について、開発者は一切の責任を負いません
  • 本アプリはユーザーの利益を保証するものではありません

10. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やアプリの機能追加に伴い、予告なく変更される場合があります。重要な変更がある場合は、アプリ内で通知いたします。

11. お問い合わせ

本プライバシーポリシーに関するご質問やご意見がございましたら、以下までご連絡ください：

  • GitHub Issues: https://github.com/hide1114/juggler/issues

12. 同意

本アプリを使用することにより、本プライバシーポリシーに同意したものとみなされます。
